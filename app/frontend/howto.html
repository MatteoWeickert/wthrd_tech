How-To

Einführung

STAC (SpatioTemporal Asset Catalog) bietet eine standardisierte Möglichkeit, Geodaten zu katalogisieren und darauf zuzugreifen. Mit der Python-Bibliothek PySTAC kann diese Struktur leicht durchsucht und verarbeitet werden. Hier erklären wir Schritt für Schritt, wie man mit PySTAC arbeitet.

Grundlegender Code

Unten finden Sie ein Beispielskript, das zeigt, wie Sie eine Verbindung zu einem STAC-Katalog herstellen, auf Collections und Items zugreifen und spezifische Daten abfragen können:


    import json
    import pystac
    import requests
    import pystac_client
    from pystac_client import Client
    
    url1 = "http://localhost:8000"
    url2 = "https://ai-extensions-stac.terradue.com/"
    
    try:
        catalog = Client.open(url1)
        print("Verbindung zu STAC-API erfolgreich!")
    except Exception as e:
        print(f"Fehler beim Verbinden mit der STAC-API: {e}")
    
    # Hole die Collection mit der ID "MLM_Collection"
    col1 = catalog.get_child("MLM_Collection")
    
    # Hole alle Items in der Collection
    # items = col1.get_all_items()
    
    # for item in items:
    #   print(f"Item ID: {item.id}")
    
    # Collection abrufen
    collection = catalog.get_child("MLM_Collection", "MLM_Collection_2")
    
    items = []
    # Durch alle Items in allen Collections iterieren
    for collection in catalog.get_children():
        print(f"Collection ID: {collection.id}")
        for item in collection.get_all_items():
            print(item.id)
            if item.id == "solar_satlas_sentinel2":
                items.append(item)
    
    # Ausgabe der Items
    for item in items:
        print(item.assets)

Schritt-für-Schritt-Erklärung

1. Verbindung zu einem STAC-Katalog herstellen

Mit Client.open() stellen Sie eine Verbindung zu einer STAC-API her. Sie können unterschiedliche URLs verwenden, um auf verschiedene Kataloge zuzugreifen:


    url1 = "http://localhost:8000" 
    url2 = "https://ai-extensions-stac.terradue.com/"
    catalog = Client.open(url1)

2. Eine spezifische Collection abrufen

Sie können eine Collection anhand ihrer ID abrufen. Hier holen wir eine Collection mit der ID "MLM_Collection":


    col1 = catalog.get_child("MLM_Collection")

3. Items aus der Collection abrufen

Nach dem Abrufen einer Collection können Sie alle enthaltenen Items durchsuchen:


    items = col1.get_all_items()
    for item in items:
        print(f"Item ID: {item.id}")

4. Iteration durch Collections und Abfrage eines spezifischen Items

Durch Iteration durch alle Collections und Items können Sie nach spezifischen Daten suchen:


    items = []
    for collection in catalog.get_children():
        print(f"Collection ID: {collection.id}")
        for item in collection.get_all_items():
            print(item.id)
            if item.id == "solar_satlas_sentinel2":
                items.append(item)

5. Assets eines Items abrufen

Nachdem Sie ein spezifisches Item gefunden haben, können Sie dessen Assets anzeigen:


    for item in items:
        print(item.assets)

Zusammenfassung

Mit PySTAC können Sie einfach auf STAC-Kataloge zugreifen, Collections und Items abrufen sowie spezifische Daten finden. Probieren Sie den Code aus, passen Sie ihn an Ihre Bedürfnisse an, und entdecken Sie die Möglichkeiten von STAC!
