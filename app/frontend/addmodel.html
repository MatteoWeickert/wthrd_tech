Modell hinzufügen
Willkommen auf der Seite "Modell hinzufügen" – Fügen Sie neue Items dem STAC-Katalog hinzu!
Auf der Seite "Modell hinzufügen" haben Sie die Möglichkeit, neue Items zum STAC-Katalog hinzuzufügen.
Ein STAC-Katalog (SpatioTemporal Asset Catalog) dient dazu, raumzeitliche Daten wie Satellitenbilder, maschinelles Lernen oder andere Geodaten effizient zu organisieren und bereitzustellen. Mithilfe des unten ausfüllbaren Formulars können Sie Datenobjekte direkt in den Katalog integrieren.
Tragen Sie die relevanten Informationen zu Ihrem Item in die dafür vorgesehenen Felder ein. Dazu gehören beispielsweise:
  • ID: Eine eindeutige Identifikation für das Item.
  • Beschreibung: Eine detaillierte Beschreibung des Items.
Die vollständige Ansicht der erforderlichen Einträge ist weiter unten sichtbar.
Was ist zu beachten?
Stellen Sie sicher, dass alle erforderlichen Felder korrekt ausgefüllt sind. Fehlerhafte Eingaben können dazu führen, dass das Item nicht korrekt im Katalog angezeigt wird.
Sobald alle Eingaben gemacht wurden, klicken Sie auf den "Analysieren"-Button. Es werden alle Felder geprüft. Kommt keine Fehlermeldung klicken sie auf "Abschicken", um das Item dem Katalog hinzuzufügen. Das Item wird dann dem Katalog hinzugefügt und kann im Modellkatalog abgerufen werden.
Sollte ein Fehler auftreten (z. B. aufgrund unvollständiger Eingaben), wird Ihnen eine Fehlermeldung angezeigt. Bitte korrigieren Sie die entsprechenden Felder und versuchen Sie es erneut. Bei Problemen mit der Verbindung zum STAC-Server stellen Sie sicher, dass Ihre Netzwerkverbindung aktiv ist.
